# D3QN 长方体高度快速参考表

## 🎯 直接使用 PPT 内置 3D 长方体的高度设置

如果你用 PPT 现成的 3D 长方体，只需要调整**高度**就能体现维度的差异。

### 📏 高度对照表（单位：像素）

| 数据形状 | 维度 | 推荐高度 | 宽度 | 说明 |
|---|---|---|---|---|
| **输入层** | 400×5 | **100px** | 80px | 基准高度 |
| **MGCN 第1层** | 400×64 | **220px** | 80px | 最高的处理层 |
| **MGCN 第2层** | 400×32 | **160px** | 80px | 中等高度 |
| **融合前（池化后）** | 32 | **160px** | 70px | 同 MGCN 第2层 |
| **融合后** | 64 | **220px** | 80px | 同 MGCN 第1层 |
| **Value 隐藏层** | 128 | **290px** | 60px | 很高 |
| **Value 输出** | 1 | **80px** | 60px | 最矮 |
| **Advantage 隐藏层** | 128 | **290px** | 60px | 很高 |
| **Advantage 输出** | 179 | **350px** | 60px | 最高 |
| **最终 Q 值** | 179 | **350px** | 80px | 最终输出 |

---

## 🔧 如何在 PPT 中设置

### Mac 版 PowerPoint
1. 选中 3D 长方体
2. 右键 → **Format Shape**
3. 找到 **Size & Properties** 标签
4. 在 **Height** 字段输入上表中的像素值
5. **Width** 和 **Depth** 保持默认或按表中设置

### Windows 版 PowerPoint
1. 选中 3D 长方体
2. 右键 → **Format Shape**
3. 在 **Size** 部分设置 **Height**
4. 输入上表中的像素值

## 📝 使用方式

1. **在 PPT 中插入 3D 长方体**
   - Insert → 3D Models → 搜索 "Cube" 或 "Box"

2. **按照上表调整每个长方体**
   - 选中长方体 → 右键 → Format Shape
   - 设置 Height 为对应的像素值
   - 设置 Fill Color 为对应的颜色

3. **用箭头连接各个长方体**
   - Insert → Shapes → Connectors
   - 从一个长方体中心连到下一个

4. **添加文字标注**
   - 在每个长方体下方或旁边添加文本框
   - 标注维度信息（如 "400×5" 或 "64"）
